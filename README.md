# ZSH Configuration

This repository contains a collection of ZSH scripts and configurations.

## Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/espocia/zsh-config.git ~/.config/zsh
    ```

2.  **Run the setup script:**

    ```bash
    cd ~/.config/zsh
    chmod +x setup.sh
    ./setup.sh
    ```

## Usage
